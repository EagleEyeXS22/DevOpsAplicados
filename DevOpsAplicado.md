## 🔧 Automatización

### 📌 Conceptos Clave
- CI/CD pipelines
- Herramientas: Jenkins, GitHub Actions, GitLab CI

### 🧠 Recordar
> Automatiza lo que se repite. Documenta lo que duele.

### 📎 Usar cuando:
- Se está integrando una nueva rama de desarrollo
- Se quiere mejorar la entrega continua
- Hay fricción entre desarrollo y QA
